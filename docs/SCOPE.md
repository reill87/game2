# 범위 — Non-goals & v1 버티컬 슬라이스

## 1. Non-goals (하지 않음)

- 실시간 멀티, PvP, 길드  
- 인앱 결제, 광고 SDK  
- 풀 보이스, 대형 오픈월드  
- 동기화 클라우드 세이브 (v1) — localStorage만  
- **검 강화식 단일 버튼 확률 강화**를 메인 루프로 유지하지 않음 (서브 이벤트로의 편입은 v2 검토)  
- **턴제 RPG 전투**를 v1 메인 루프로 두지 않음 (경영 시뮬 본체)

## 2. v1 버티컬 슬라이스 (플레이 가능 최소 단위)

**한 줄:** 새 게임 시작 → **첫 작품** 튜토리얼(G1+T1, 배치·주간 틱·출시 패널) → **두 번째 작품** 기획부터 Appeal·홍보 해금 → **사무실 2단계** 업그레이드까지 또는 2작 출시 후 엔딩 크레딧(간이).

상세 루프·수치: [PRODUCT_LOOP.md](./PRODUCT_LOOP.md), [BALANCE.md](./BALANCE.md).

### 포함해야 할 것

- [ ] **배치 화면** + 오배치 패널티 ([BALANCE.md](./BALANCE.md))  
- [ ] **주간 개발 틱** (Progress / BugDebt, Appeal은 2작부터)  
- [ ] **출시 패널** (튜토리얼: 출시 + 1주 폴리싱만; 2작~ 홍보 단계)  
- [ ] **결과 화면** (리뷰·매출·골드) + **localStorage** 저장  
- [ ] **사무실 3단계 중 1→2 업그레이드** 1회 (고용 한도·하드 캡 연동)

### v1에서 생략 가능

- 클라우드 세이브, 리더보드  
- 직원 다수·랜덤 이벤트 대량, 연구 트리 대형화  
- 장르·테마 **3×3** 초과 확장
