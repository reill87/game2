# UI · 씬 플로우 (초안)

## 1. Phaser 씬 후보

| 씬 키 | 역할 |
|--------|------|
| Boot | 설정 로드, 폰트, 저장 마이그레이션 |
| Title | 타이틀, 새게임/이어하기 |
| Hub | 팀 상태, 과제 목록, 상점(간이) |
| MissionPrep | 파티 편성, 장비, 소모품 |
| Battle (또는 Mission) | 턴 진행, 로그, 스킬 선택 |
| Result | 승패, 보상, 대사 |
| Pause | 설정, 종료 |

v1에서 **Hub + MissionPrep 통합** 가능하면 씬 수 줄이기.

## 2. 화면 와이어 포인트 (세로 720×1280 기준)

- **상단**: 재화, 현재 과제명, 일시정지  
- **중앙**: 캐릭터 초상(또는 실루엣) + 적 리스트  
- **하단**: 행동 버튼 (공격 / 스킬 / 아이템 / 방어)  
- **로그**: 최근 3~5줄 스크롤 또는 페이드

## 3. 입력

- 터치 우선  
- 키보드 단축키: `SPACE` 확인, `1~4` 스킬 슬롯 (선택)

## 4. 접근성·UX

- 텍스트 속도 슬라이더 (나중에)  
- 연출 스킵: 길게 누르기 또는 더블 탭  
- 색만으로 중요 정보를 구분하지 않기 (아이콘 병행)

## 5. 기존 game 과의 차이

- `GameScene` 단일 거대 파일 지양 → **BattleUI / HubUI 클래스 분리** 목표  
- 모달은 씬 전환 대신 **Container 오버레이**로 통일할지 규칙 정하기
