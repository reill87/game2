# 시스템 명세 (미니 GDD 초안)

## 1. 직군 (클래스)

| 직군 | 역할 (전투 은유) | 기본 역할 |
|------|------------------|-----------|
| 기획 | 스펙/우선순위 | 버프, 자원 회복, 적 “스코프” 감소 |
| 디자인 | UX/비주얼 | 방어, 쉴드, 적 공격 패턴 완화 |
| 개발 | 구현 | 딜링, 버그 “처치”, 기술 부채 정리 |

v1에서는 **스킬 2~3개/직군**만 두고 확장한다.

## 2. 성장

- **레벨**: XP 누적 → 최대 레벨은 v1에서 낮게 캡 (예: Lv 10)  
- **스탯 (초안)**: HP, 집중력(MP), 공격력, 방어력, 속도(선택)  
- **스킬 해금**: 레벨 또는 스토리 플래그 — v1은 레벨만 사용 가능  
- **전직/명성**: v1 비포함, v2 메모만

## 3. 과제(던전) — “전투” 규칙 초안

### 3.1 단위

- **턴**: 플레이어 페이즈 → 적(이슈/마감) 페이즈  
- **행동 예시**: 일반 처리(공격), 스킬, 아이템(커피·문서화), 방어  
- **승리 조건**: 적 HP 0 또는 “진행도 100%”  
- **패배 조건**: 파티 전원 기절 또는 “데드라인 0”

### 3.2 적 타입 (은유)

- **버그**: 높은 딜, 낮은 HP  
- **스코프 크립**: 방어 높음, 턴 지연 유발  
- **레거시 덩어리**: 광역 피해, 정리 스킬에 약점 (v2)

### 3.3 직군 시너지 (초안)

- 기획 + 개발: 첫 턴 집중력 +10%  
- 디자인 + 기획: 방어 관통 +5%  
- 3직군 풀파티: 클리어 시 보너스 골드 (v1 선택)

## 4. 경제

- **골드**: 과제 보상, 장비 강화(단순 +스탯) 또는 소모품 구매  
- **재료**: v1 생략 가능 — 있으면 “티켓 클로즈 보상” 1종만  
- **현금 과금**: 없음

## 5. 장비 (v1 최소)

- 슬롯: 무기(도구) / 방어구(노트북·모니터?) / 악세(명함?) — 이름만 풍자  
- 효과: 평탄한 스탯 +1~3 수준, 세트 효과는 v2

## 6. 저장에 넣을 데이터 (초안)

```text
버전, 팀원별 {직군, 레벨, XP, 스킬장착, 장비},
보유 골드/아이템,
스토리/과제 클리어 플래그,
옵션(볼륨, 텍스트 속도)
```

상세 스키마는 구현 단계에서 `SaveSystem`으로 정리.
